Procédure d'utilisation du framework GabriX.

Dans la console dans le dossier dans lequel créer le projet :

Création d'un nouveau projet -> GabriX install
Création d'une base de donnée -> GabriX create:database
Création de l'entité User et du mécanisme de session -> GabriX create:user
Création d'entité de la bdd -> GabriX make:entity

Dans le cas de l'utilisation de Sass, ne pas oublier d'entrer la commande npm run sass-dev dans la console

Twig 3.0 est utilisé pour la génération des templates, toutes les informations -> https://twig.symfony.com/doc/3.x/intro.html#installation
